# Простой калькулятор 

## О проекте
Простой калькулятор на Python для выполнения базовых математических операций. Проект разработан в учебных целях для демонстрации навыков работы с функциями. 

## Функциональность
- ➕ **Сложение** - сложение двух чисел
- ➖ **Вычитание** - вычитание двух чисел
- ✖️ **Умножение** - умножение двух чисел
- ➗ **Деление** - деление двух чисел с проверкой на ноль

## Требования/зависимости
- Python 3.8 или выше
- Внешние библиотеки не требуются (используются только стандартные модули Python)

## Установка
```bash
# Клонирование репозитория
git clone https://github.com/Damian-17/calculus_tool.git
cd calculus_tool

# Создание виртуального окружения
python -m venv venv

# Активация виртуального окружения
# Для Windows:
venv\Scripts\activate
# Для Mac/Linux:
source venv/bin/activate

# Запуск проекта:
py calculator.py
